fdisk /dev/vdb

欢迎使用 fdisk (util-linux 2.23.2)。 更改将停留在内存中，直到您决定将更改写入磁盘。 使用写入命令前请三思。

Device does not contain a recognized partition table 使用磁盘标识符 0xec7feac9 创建新的 DOS 磁盘标签。

命令(输入 m 获取帮助)：n Partition type: p primary (0 primary, 0 extended, 4 free) e extended Select (default p): p 分区号 (1-4，默认 1)：1 起始 扇区 (2048-104857599，默认为 2048)：回车 将使用默认值 2048 Last 扇区, +扇区 or +size{K,M,G} (2048-104857599，默认为 104857599)：回车 将使用默认值 104857599 分区 1 已设置为 Linux 类型，大小设为 50 GiB

命令(输入 m 获取帮助)：w The partition table has been altered!

Calling ioctl() to re-read partition table. 正在同步磁盘。
