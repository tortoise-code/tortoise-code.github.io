# 安装配置IJ编辑器

## 下载的IntellJ Idea正常安装

## 激活工具内部有教程 

> 放在桌面双击文件夹内后缀名`vbs`文件


## IntellJ 打开项目

`物流管理`目录改成英文名称,例如`warehouse`

## IntellJ 安装中文插件

1. 打开`file`或`文件`

![/ijzh.jpg](http://media.codecore.cn/markdown/ijzh.jpg)

2. 选中`setting`

3. 找到`Plugin`或者`插件`

4. 输入`chinese`安装简体中文

![/ijzh1.jpg](http://media.codecore.cn/markdown/ijzh1.jpg)
