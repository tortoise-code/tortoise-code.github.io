# 服务器环境搭建

宝塔面板网站选中java项目

会提示安装JDK或tomcat

由于linux系统无法直接使用exe,所以linux版的jdk可以通过先安装tomcat,使用tomcat内置的java jdk

  1. 选择安装tomcat
  2. 安装tomcat8
  3. 等待安装完成

  1. tomcat8安装完毕后

  2. 刷新页面,点击JDK管理

一般安装完毕tomcat8默认会生成jdk的配置,如上图

如果默认生成的jdk地址不是java后缀,如:/usr/local/btjdk/jdk8/bin/java 点击添加jdk在该地址目录找到java,选中它,完成添加
